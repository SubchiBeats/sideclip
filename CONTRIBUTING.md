# Contributing to SideClip

Thanks for helping improve SideClip.

## Development

SideClip requires Node.js 20 or newer and has no runtime dependencies.

```bash
git clone https://github.com/SubchiBeats/sideclip.git
cd sideclip
npm start
```

Run the test suite before opening a pull request:

```bash
npm test
```

## Project principles

- Keep the default experience useful without paid services.
- Preserve user privacy and self-hosting as first-class features.
- Prefer browser and Node.js platform APIs over new dependencies.
- Be honest about platform limitations and avoid compliance guarantees.
- Add focused tests for changes to authentication, storage, uploads, or export.

## Pull requests

Keep pull requests scoped and explain the user-facing outcome. Include manual
verification notes for visual or media-rendering changes.
